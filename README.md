# DockerHubSecretScraper
Scans recently updated public docker hub images for secrets

### Secrets it looks for 
"Slack Token" 
"RSA private key"  
"SSH (OPENSSH) private key"  
"SSH (DSA) private key"  
"SSH (EC) private key"   
"PGP private key block"  
"Facebook Oauth"  
"Twitter Oauth" 
"GitHub"  
"Google Oauth"  
"AWS API Key"  
"Heroku API Key"  
"Generic Secret"   
"Generic API Key"  
"Slack Webhook"   
"Google (GCP)"  
"Twilio API Key"   
"Password in URL"
